# PITFALL! Recovered Unity Project

This repository contains a recovered Unity project reconstructed from an old `PITFALL!` Android APK for preservation, compatibility repair, and modernization work.

What this repo is:

- a salvaged Unity project that opens in Unity `6000.4.1f1`
- recovered scenes, assets, shaders, scripts, and project settings
- a working base for continued compatibility and visual/audio fixes

What this repo is not:

- the original upstream Activision source repository
- a perfect byte-for-byte recreation of the original project

Project entry point:

- open this folder directly in Unity Hub
- start with `Assets/Scenes/States/Boot.unity`

Notes:

- this project was recovered from an APK, so some systems have been reconstructed and may still need cleanup
- generated Unity folders such as `Library`, `Temp`, and `Logs` are intentionally excluded from version control
